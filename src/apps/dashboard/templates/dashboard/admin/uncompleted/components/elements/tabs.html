ناوبری زبانه ها

از افزونه جاوااسکریپت زبانه بوت استرپ برای گسترش زبانه و قرص های ناوبری برای ایجاد صفحه های زبانه دار از محتوای لوکال، حتی از طریق لیست های کشویی استفاده کنید.

مثال های پایه

شما به سادگی می توانید با مشخص کردن data-bs-toggle="tab" یا data-bs-toggle="pill"، ناوبری یک زبانه یا قرص را بدون نوشتن هیچ جاوااسکریپتی فعال کنید. یک عنصر در اینجا برخی از کامپوننت های پایه زبانه وجود دارد که می توانید از آن ها استفاده کنید.
