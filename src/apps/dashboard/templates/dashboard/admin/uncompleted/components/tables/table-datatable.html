مثال جدول داده ها

با استفاده از جدول داده ها، کنترل های تعاملی پیشرفته را به جداول HTML خود اضافه کنید. این یک ابزار بسیار انعطاف پذیر است و تمام امکانات پیشرفته به شما اجازه می دهند تا جدول را به صورت فوری و زیبا نمایش دهید.

برای یک نمای کلی، مستندات را بررسی کنید.
